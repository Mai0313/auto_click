歡迎來到 Auto Click 專案。

## 概述

本專案提供自動點擊開發環境。包含配置和執行自動點擊序列的功能。

## 安裝

1. 複製儲存庫。
2. 導航至專案目錄。
3. 按照文件中的設置說明進行操作。

## 使用方法

- 按照提供的說明執行主應用程式。
- 根據需要配置設定。
